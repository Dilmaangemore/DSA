Take an integer N as input from user, then take N characters as input. And print the Frequency of Each Character.

Input Format

Integer N
All ASCII characters
Constraints

1<=N<=10^5
Output Format

Frequency of each character.

Sample Input 0

6
a b a d b c
Sample Output 0

a 2
b 2
c 1
d 1
Sample Input 1

7
* / - + * . +
Sample Output 1

* 2
+ 2
- 1
. 1
/ 1
